# 陕西话（陕西方言）研究手册

> 面向「Claude Code 方言对话插件」的语料整理，目标：让 AI 在文字聊天中用地道陕西话与陕西用户对话。涵盖语音、词汇、语法、句式、习俗与对话示例。

---

## 1. 概述

陕西话是流行于陕西省境内、以关中片（西安、咸阳、宝鸡、渭南、铜川）为核心、兼及陕北晋语与陕南西南官话的中原官话分支。狭义的"老陕话"主要指**关中方言**，属于中原官话关中片-秦陇片，是周秦汉唐以来的"雅言"嫡系后代。

陕西话最大的特征是**古音保留多**：保留了全浊声母、入声字（虽调值已变，但常独立成调类）、尖团音，部分地区分前后鼻音但不严格。音调上常用四声加一个"轻声化"的入声尾，听起来高亢、硬朗、干脆。

性格气质上，陕西话自带一种**"生冷蹭倔"**的味道：说话不拐弯抹角，直来直去，不喜客套，语速不慢但字字带劲。"谝闲传""咥饭""嘹得很""咱""弄啥咧"……这些词不是装出来的文雅，而是几千年关中老农、城墙根下老汉、回民街坊、秦腔戏班子一代代传下来的活语料。

陕西人说话像"吼秦腔"，声音冲、节奏稳、尾音短促，常用"咧""哩""嘞""嘛""呀"等语气词把一句话钉死。一句"咱走，咥面去！"，比普通话"咱们走，去吃面"有力十倍。

## 2. 语音特征

### 2.1 声调

陕西话（关中片）单字调一般分四声，但**入声字多独立成调**或并入阴平，听感上比普通话多一份"短促硬收"的味道。

| 调类 | 例字 | 大致调值 | 备注 |
|------|------|----------|------|
| 阴平 | 高、天、三 | 31/21 | 低降起，听感比普通话低 |
| 阳平 | 来、白、同 | 24 | 中升 |
| 上声 | 老、好、有 | 53 | 高降，尾音硬 |
| 去声 | 走、是、菜 | 44 | 中平偏稳 |
| **入声（古）** | 八、吃、麦、桌、骨 | 短促 | 部分地区并入阴平，但发音短促，常带喉塞尾 |

注意：**声调走向与普通话差异极大**——普通话阴平是高平(55)，陕西话阴平多是低降(31)，所以外地人听陕西人说话，会觉得"调子低、平、快、硬"。

### 2.2 声母特征

1. **不分平翘舌（zh/ch/sh 与 z/c/s）**：这是最显著的标志。
   - "知道"读如"资道"（"知"≈zi）
   - "吃饭"读如"次饭"
   - "山"≈"三"、"师"≈"思"
   - 但西安城区年轻人已部分恢复平翘区分，老一辈关中农村话更明显。

2. **不分前后鼻音（-n / -ng）**：
   - "陈"≈"程"、"林"≈"灵"、"金"≈"京"
   - "人"≈"仍"、"根"≈"耕"

3. **保留古声母**：
   - 部分字保留中古"疑"母（ng-）："我"读 ngě；"饿""恶"常带 ng 声母。
   - "白""爬""病"等古全浊声母字，陕西话声母常送气更明显。

4. **f / h 偶尔相混**：
   - "飞"≈"灰"、"放"≈"晃"在部分关中片村话里出现。

5. **r 声母字**：
   - "热"读如"叶"（y-）或"ze"；"人"读如"银 yín"；"软"读如"van"。

### 2.3 韵母特征

1. **"街"读如"该"gāi** —— 主要元音开口度大。
2. **"鞋"读如"孩"hái**。
3. **"解"（解开）读 gǎi**，"解开"≈"该开"。
4. **"咸"（盐）读 hán**（西安话），"咸菜"≈"含菜"。
5. **"核"（核桃）读 hú**。
6. **"很"读 hěn** 但偏后，听感接近"恨"。
7. **"给"读 jǐ**（介音保留）。
8. **"国"读 guěi** 或 "guǐ"，保留入声韵。
9. **"白"读 bé 或 piě**，"百"读 bé。
10. **"北"读 běi 或 bě**，"墨"读 mèi 或 mé。
11. **"说"读 shuě 或 shuó**（介音模糊）。
12. **"水"读 fěi**（部分县份，嘴/唇音转换）。
13. **"没"读 mú**（常用语："有木？"）。

整体韵母更偏开口呼、洪音化，听起来"圆、钝、宽"，少尖细感。

## 3. 特色词汇与俚语

> 标注：★ = 极高频核心词，必须掌握；◆ = 地域/口音差异；☆ = 古语/书面化但仍用。

### 3.1 核心动词

- ★**咥（diě）**：吃。"咥饭""咥面""咥肉"——生猛、带劲地吃，关中人不说"吃饭"，说"咥饭"。
- ★**谝（piǎn）**：闲谈、聊天。"谝闲传""谝干传""谝活""谝大空"。
- ★**撩（liáo）**：调戏、招惹、招惹是非。"撩咂咧""撩是撩非"。
- ★**嘹（liáo）**：好、棒、漂亮。"嘹得很""嘹咋咧"（好得很）。
- ★**弄（nèng）**：万能动词，做/搞/弄/整。"弄啥咧？"
- ★**攒（zǎn）**：积攒、攒劲。"攒劲得很"=很厉害。
- ★**得是（děi si）**：是不是（询问）。"你得是想走？"
- ★**咋（zǎ）**：怎么。"咋咧？""咋办？"
- ◆**蹚（tāng）**：蹚水、蹚路（探路）。
- ☆**劬（qú）**：辛苦（旧字，今口语较少）。

### 3.2 常用名词 / 称谓

- ★**咱（zán）**：我、我们。陕西人自称不用"我"或"俺"独尊，常用"咱"。
- ★**达达（dá da）**：爸爸（关中、陕北普遍）。
- ★**nia nia / 娘娘（niǎ niǎ）**：奶奶（关中话）。
- ★**爷（yé）**：爷爷 / 父亲（关中部分县份"爷"指父亲）。
- ★**叔（shú）**：叔叔（入声保留）。
- ★**姑（gū）**：姑姑。
- ★**外爷 / 外婆**：外公外婆。
- ★**碎娃（suì wá）**：小男孩。
- ★**碎女子（suì nǚ zi）**：小女孩。
- ★**后生（hòu sheng）**：年轻小伙子。
- ★**婆姨（pó yi）**：妻子、媳妇。"我屋里的婆姨"。
- ★**掌柜的**：丈夫、当家的。"咱掌柜的"。
- ★**屋里的**：妻子（"屋"读 wǔ，入声）。
- ★**老汉**：老头、老公（亲昵）。
- ★**乡党（xiāng dang）**：老乡。
- ★**瓜娃 / 瓜皮 / 瓜不兮兮（guā bù xī xī）**：傻、愣、笨（骂人/亲昵用）。
- ★**二杆子 / 二球 / 半吊子**：愣头青、二愣子。
- ★**怂（sóng）**：笨、差劲、丢人。"怂包""怂得很"。
- ★**锤子 / 球 / 蛋**：表示"啥也不是"、"没用"。"锤子！"= "啥也不是！""球！"= 去你的！
- ★**乡党**：老乡、乡亲。

### 3.3 食物 / 物件（关中特有）

- ★**biángbiáng 面**：陕西名面，biáng 字为专用字（写法复杂，常以"biangbiang"代之）。
- ★**臊子面**：岐山臊子面，红油、酸汤、漂菜。
- ★**肉夹馍**：白吉馍夹腊汁肉（"中国式汉堡"）。
- ★**油泼面**：宽面、辣面、油泼辣子。
- ★**甑糕（zèng gāo）**：糯米红枣蒸糕。
- ★**凉皮**：面皮/米皮。
- ★**葫芦头**：大肠泡馍。
- ★**羊肉泡馍**：自己掰馍、老汤煮。
- ★**浆水面**：浆水菜下面。
- ★**石子馍**：石子上烙的饼。

### 3.4 形容词 / 口头禅

- ★**嘹（liáo）**：好、棒。"嘹得很""嘹咋咧"。
- ★**攒劲（zǎn jìn）**：厉害、得劲。
- ★**克里马擦（kè li mǎ cā）**：麻利、快、赶紧。"克里马擦走！"
- ★**撩（liáo）**：调戏、惹。
- ★**生冷蹭倔**：倔强、不服软。"这娃生冷蹭倔得很！"
- ★**瓷（cì）**：硬、扎实。"这瓜瓷得很"=这瓜很硬/扎实。
- ★**烂怂（làn sóng）**：差劲、不好。
- ★**呆（dāi）**：傻、愣。"你瓜不兮兮呆啥咧？"
- ★**谝闲传**：闲聊天。

### 3.5 指示 / 方位

- ★**兀达 / 兀搭（wù da）**：那里。
- ★**这达 / 这搭（zhè da）**：这里。
- ★**哪达（něi da / nǎ da）**：哪里。"你哪达人？"=你是哪里人？
- ★**跟（gēn）**：和。"我跟你说话"（语气更直）。

### 3.6 数字 / 时间

- **一搭里（yì dā li）**：一起。
- **一满（yì mǎn）**：全部、整个。
- **夜个（yè ge）**：昨天。
- **前个 / 前儿个（qián ge）**：前天。
- **明儿 / 明个（míng ge）**：明天。
- **年时（nián si）**：去年。
- **今儿 / 今个（jīn ge）**：今天。
- **黑咧 / 黑里（hēi lie）**：晚上。

## 4. 语法特征

### 4.1 语气词系统

陕西话的语气词是其灵魂，比普通话丰富得多，常用：

- **咧（lie）**：表状态持续、感叹、疑问。"走着咧""咋咧？"
- **哩（li）**：相当于"呢/的"，"弄啥哩？"
- **嘞（lei）**："我嘞个神嘞！"
- **嘛（ma）**：催促、肯定。"走嘛！"
- **呀（ya）**：感叹。"好呀！"
- **得（de）**：表状态。"他走得了。"
- **着（zhe）**：进行态。"吃着咧""走着咧"。

### 4.2 句法差异

1. **"咱"作主语/领属**：不仅表"我们"，也表"我"。
   - "咱走！"
   - "咱屋达达"= 我爸爸
   - "咱妈"= 我妈

2. **"弄"作万能动词**：
   - "弄啥咧？"= 干啥呢？
   - "你弄个啥？"= 你搞啥？
   - "弄俩菜"= 做两道菜

3. **AA 式形容词重叠**：
   - "高高""大大""红红"（不叠 AABB）。
   - "美得很"（"很"后置）。

4. **副词后置**：
   - "你先走"→"你先走嘹"
   - "美得很""嘹得很""攒劲得很"

5. **完成态/持续态**：
   - "咥过"= 吃过了
   - "走着咧"= 正在走
   - "了"常读 liao 轻声。

6. **被动 / 处置**：
   - "叫"代替"被"。"叫狗咬了"
   - "把"常省略："碗打咧"= 把碗打了

7. **比较句**：
   - "A 比 B + 形容词 + 得 + 多" → 关中常说"A 比 B 强多了"
   - "他不跟我一样"→"他不跟咱一样"

8. **反复疑问**：
   - "咋？咋？咋咧？"= 怎么啦？
   - "得是？得是？"= 是吗是吗？

9. **量词用法**：
   - "个"字使用极广，"一搭里""一满""一伙子"。

10. **亲属称谓作呼语**：
    - "达达""爷""叔""哥"可当面称呼亲属或非亲属（乡里乡亲）。

## 5. 对话句式（20+ 常用）

1. "弄啥咧？" = 你在干啥？
2. "你得是走呀？" = 你要走吗？
3. "咱走，咥面去！" = 咱走，吃面去！
4. "嘹得很！" = 好得很！
5. "攒劲得很！" = 太厉害了！
6. "怂得很！" = 太差劲了！
7. "瓜娃！" = 傻小子！
8. "锤子！" = 去你的！
9. "咋咧咋咧？" = 怎么啦？
10. "克里马擦！" = 快点！
11. "走，咱谝会儿闲传。" = 走，咱聊会儿天。
12. "你屋达达哩？" = 你爸呢？
13. "今黑咧吃啥？" = 今晚吃啥？
14. "一搭里走！" = 一起走！
15. "兀达咋？" = 那里怎么了？
16. "你哪达人？" = 你哪里人？
17. "把票子给咱。" = 把钱给我。
18. "美得很！" = 太美了/好得很！
19. "你甭（beng）去。" = 你别去。
20. "甭给咱说。" = 别跟我说。
21. "咱屋掌柜的不在。" = 我家老公不在。
22. "叫狗咬了。" = 被狗咬了。
23. "走嘛，再不走就迟了。" = 走啊，再不走就晚了。
24. "这达兀达都是咱的。" = 这里那里都是我的。
25. "咱屋里坐。" = 来我家坐。
26. "你喝汤了没？" = 你吃饭了没？
27. "甭急，急啥咧？" = 别急，急什么。
28. "我跟你说个事。" = 我给你说个事。
29. "行，就这么弄！" = 行，就这么办！
30. "碎娃，去耍！" = 小孩，去玩！

## 6. 书面表达方式

陕西话在文字聊天里要"打出秦味"，关键在于**谐音字的选择**与**语序习惯**。

### 6.1 常用谐音字

| 方言词 | 常见书写 | 说明 |
|--------|----------|------|
| 咥 | 咥 | 不要写"叠""碟" |
| 谝 | 谝 | 不要写"片""骗"（年轻网友常错） |
| 嘹 | 嘹 | 关中"好"的意思，不要写"聊" |
| 撩 | 撩 | 不要写"了" |
| 咧 | 咧 | 表语气，不要写"了" |
| 嘞 | 嘞 | 不要写"了""的" |
| 咱 | 咱 | 不要写"咱"（一致） |
| 怂 | 怂 | 不要写"松""耸" |
| 锤 | 锤 / 锤子 | 不要写"垂" |
| 瓜 | 瓜 | 不要写"挂""寡" |
| 兀 | 兀 | 不要写"那""屋" |
| 达 | 达 | 不要写"大""搭"（书面可选"达"或"搭"） |
| 婆姨 | 婆姨 | 不要写"破衣" |
| 碎娃 | 碎娃 | 不要写"岁娃" |
| 乡党 | 乡党 | 不要写"香当" |

### 6.2 文字聊天实用技巧

1. **语气词是灵魂**：一定要用"咧""哩""嘞""嘛""呀"收尾，没了语气词就没了陕西味。
2. **句首加"咱"**：陕西人开口爱说"咱"，即便只代表自己。
3. **短句硬收**：一句话不要长，4-10 字为佳，节奏感强。
4. **动词生猛**：用"咥""谝""撩""嘹""攒劲"代替"吃""聊""调戏""好""厉害"。
5. **用"甭"代替"别"**：陕西方言保留"甭"。
6. **用"咱"自称**：避免频繁"我""俺"。
7. **疑问句加"得是"**："你得是……？"= "你是不是……？"
8. **感叹用"嘹得很"**：避免"很好""太棒了"。
9. **亲昵称谓**：对话里可称呼"乡党""碎娃""后生""掌柜的"。
10. **古语感**："咥""甭""得是""兀达"……自带古语味道。

### 6.3 注意事项

- 不要把"谝"写成"骗"，会把"聊天"变成"行骗"。
- 不要用太多"滴""哒""哟"等网红词，那不是陕西话，是"东北话+网红话"。
- 不要叠用"呀呀呀""啊啊啊啊"——陕西话是生猛简洁的，不是嗲。

## 7. 典型对话示例

### 场景一：西安城墙根下大爷聊天（日常闲谝）

**陕西话：**
> 大爷甲：乡党，今儿个天爷好着呢，晒得人暖和。
> 大爷乙：是嘞，克里马擦出来谝闲传。
> 大爷甲：你屋达达哩？咋没见他出来耍？
> 大爷乙：我达达屋里歇着咧，前儿个闪了腰，今儿个走不利索。
> 大爷甲：唉，老咧，不比后生那阵了。你屋后生呢？
> 大爷乙：后生上班去咧，在外头挣钱，屋就咱俩老的。
> 大爷甲：咱也美了，娃们有出息，咱享福。
> 大爷乙：享啥福？黑咧还得咥饭、走达走达，没清闲。
> 大爷甲：哈哈，你这人，攒劲得很！
> 大爷乙：锤子！走，咱去回民街咥碗泡馍！
> 大爷甲：嘹得很！走嘛！

**普通话对照：**
> 大爷甲：老乡，今天天气真好，晒得人暖和。
> 大爷乙：是啊，麻利点出来聊天。
> 大爷甲：你爸呢？怎么没见他出来玩？
> 大爷乙：我爸在家里歇着，前几天扭了腰，今天走路不利索。
> 大爷甲：唉，老了，不像年轻人那阵了。你儿子呢？
> 大爷乙：儿子上班去了，在外面挣钱，家里就我俩老人。
> 大爷甲：咱们也享福了，孩子们有出息，咱们享福。

> 大爷乙：去你的！走，咱们去回民街吃碗泡馍！
> 大爷甲：好得很！走！

## 8. 民间习俗与里程碑事件表达

### 8.1 关中民间文化

1. **秦腔**：陕西的魂。"吼秦腔"——不是唱，是吼。生、猛、硬、冲。
   - "今儿个咱去易俗社听秦腔！"
   - "他那嗓子，吼秦腔攒劲得很！"
   - "不会吼两句秦腔，算啥老陕？"

2. **biángbiang 面**：最陕西的面。
   - "走，咥碗 biángbiang 面！"
   - "老陕不吃 biángbiang 面，等于没来陕西。"
   - biáng 字写法："一点飞上天，黄河两道弯，八字大张口，言字往里走……"

3. **肉夹馍 / 凉皮 / 羊肉泡馍**：
   - "肉夹馍是咱陕西人的命！"
   - "凉皮三件套：面皮+豆皮+冰峰（汽水），攒劲得很！"
   - "咥泡馍自己掰馍，嘹得很！"

4. **葫芦头 / 葫芦鸡**：
   - "葫芦头泡馍，重口味！"
   - "葫芦鸡，鸡里最攒劲！"

5. **关中八大碗**：传统席面——红肉、白肉、黄焖鸡、方块肉、酥肉、条子肉、八宝饭、烩菜。

6. **社火**：
   - "芯子"——芯子里站小孩的高台社火。
   - "踩高跷"——踩着高跷走街。
   - "扭秧歌"——陕北秧歌、关中秧歌。
   - "马勺脸谱"——宝鸡凤翔特色。
   - "户县农民画"——鄠邑区农民画。

7. **皮影戏**：陕西皮影，华阴、华县最盛。"老腔皮影，吼起来攒劲得很！"

8. **关中"席面"**：结婚、丧葬、过年、过满月——八大碗为主。

### 8.2 关中婚礼习俗

1. **挂门帘**：迎亲当天，新娘到门口，挂门帘进门，婆姨嫂子们接人。
2. **认大小**：新郎新娘拜见公婆叔伯，认识家族长辈，改口叫爸妈。
3. **闹房**：晚上乡党朋友闹洞房，新郎挨"打"。
4. **三天回门**：婚后第三天，新娘回娘家。

陕西话表达：
> "明儿个咱屋后生结婚，挂门帘去！"
> "你屋女子明儿个认大小，你准备啥红包？"
> "今儿个闹房攒劲得很，把新郎官整怂了！"

### 8.3 关中白事 / 丧葬

- **白事**：老人过世。
- **唢呐**：丧礼必请，哭腔哀乐。
- **摔盆**：长子摔盆。
- **上坟**：清明、农历十月一、寒衣节上坟烧纸。

陕西话表达：
> "咱屋爷走了……呜……"
> "明儿个上坟，给咱nia nia烧点纸。"
> "唢呐一响，眼泪都下来咧。"

### 8.4 重大事件 / 时事表达

- **节日**：
  - "过年咱走亲戚，攒劲得很！"
  - "十五闹元宵，社火攒劲！"
  - "中秋咥月饼，咱屋婆姨做的攒劲！"

- **人生大事**：
  - "你屋娃考上大学咧？攒劲得很！"
  - "你屋女子出嫁咧？门帘挂好没？"
  - "咱屋盖房咧，你来帮忙！"

- **国家大事 / 时事**：
  - "新闻上说……咱国家攒劲得很！"
  - "足球比赛，中国队赢咧？锤子！"
  - "大飞机造出来咧，攒劲！"

- **骂人吐槽**：
  - "这事弄啥咧？锤子！"
  - "这人怂得很，瓜不兮兮！"
  - "这天气攒劲咧，晒得人木乱！"

### 8.5 关中老陕特有的"生冷蹭倔"

陕西人说话"生冷"——生硬冷峻，不带客气。
陕西人说话"蹭倔"——倔强不服软。
老陕说话的几个特点：

1. **不客套**：开门见山，不"您吃了没？""最近可好？"
2. **带感情**：骂人也是亲昵，锤子/怂/瓜娃——都是爱称。
3. **直接**："你想弄啥？直说！"
4. **拍胸脯**："咱这事，咱办！"
5. **爱面子**："咱屋达达是攒劲人！"

## 9. 给 AI 的使用建议

1. **"咥"代替"吃"**——动词生猛。一句"咥碗面"比"吃碗面"地道十倍。
2. **"咱"自称**——陕西人开口爱说"咱"，无论代表自己还是咱们。
3. **语气词必须挂尾**——"咧""哩""嘞""嘛"是一句话的"灵魂"，少了就没味道。
4. **句式要短、要硬**——别写长句，4-10 字最好，节奏感强。
5. **直来直去，不客套**——陕西话不要"您""请""谢谢"挂在嘴边，那不是陕西味。
6. **骂人用"锤子""瓜娃""怂"**——这些是亲昵词不是真的骂人，但别对长辈用。
7. **亲昵用"乡党""碎娃""后生""掌柜的""婆姨"**——日常称呼自带关中味。
8. **保留"咧""哩""嘞"等语气**——这些是陕西话的灵魂。
9. **古语保留**："甭""得是""兀达"等用起来——自带古语味道。
10. **不要"滴""哒""哟"等网红词**——陕西话是生猛简洁的，不要嗲。

## 10. 总结

陕西话的核心是**"生冷蹭倔"的关中老陕味**——说话直、声音冲、字字带劲、自带古音。要让 AI 学会陕西话，关键在于：

1. **动词生猛**：咥、谝、撩、嘹、攒劲——让动词有劲。
2. **语气词挂尾**：咧、哩、嘞、嘛——一句话的灵魂。
3. **自称用咱**：不"我"不"俺"，开口"咱"。
4. **短句硬收**：别长句，节奏感要强。
5. **保留古语**：甭、得是、兀达、nia nia——自带老陕味。
6. **直来直去**：不客套、不拐弯、亲昵也直接。

记住：**老陕说话像吼秦腔——不吼不痛快！**

# 上海话（吴语上海话/沪语）研究报告

> 适用对象：为「Claude Code 方言对话插件」准备的上海话素材库
> 目标：让 AI 能在文字聊天中"用地道上海话跟上海用户聊天"
> 编写日期：2026/06/23

---

## 一、概述

上海话属于**汉藏语系汉语族吴语区太湖片沪苏小片**，是吴语的重要代表。**上海话是所有中国方言中"软糯"程度最高的方言之一**——苏州话、杭州话、宁波话都比上海话更"硬"。上海话自带一种"慢、细、糯"的味道，听着像在品一壶龙井。

上海话最显著的三个字是 **"阿拉、侬、伊"**：
- **阿拉（a la）** = 我、我们。"阿拉上海宁" = 我是上海人。这是上海话最标志性的词。
- **侬（nóng）** = 你、您。比普通话"你"更礼貌。
- **伊（yī）** = 他、她、它。保留古汉语第三人称。

上海话骨子里透着一股**"小布尔乔亚 + 老克勒"**的气质：
- **小布尔乔亚** = 小资情调，喝咖啡、逛弄堂、穿旗袍
- **老克勒** = 老派上海人，穿着讲究、说话得体、懂规矩
- **上海人"讲规矩"** = 排队、加塞、付账 AA 都讲规矩
- **上海人"拎得清"** = 明白事理、不糊涂
- **上海人"精打细算"** = 不是小气，是讲实惠

**核心提示**：上海话是少数几个**听不懂普通话也能写出来**的方言——因为它有自己的"谐音字体系"：用"侬"代替"你"，用"阿拉"代替"我们/我"，用"伊"代替"他"。**AI 在文字聊天里只要把关键代词换成上海话谐音字，就已经"沪味十足"了**。

## 二、语音特征

### 2.1 声调（最显著特征：保留入声）

上海话有 **8 个声调**（实际音韵上是 7 个独立声调 + 1 个轻声变体），是吴语区声调保留最完整的方言之一。

| 调类 | 上海话调值 | 普通话对应 | 例字 |
|------|------------|------------|------|
| 阴平（一声） | 53（全降） | 高平 | 高、天、飞 |
| 阳平（二声） | 23（低升） | 中升 | 来、同、白 |
| 阴上（三声） | 34（中升） | 降升 | 古、走、好 |
| 阳上（四声） | 24（中升） | 降升 | 五、近、动 |
| 阴去（五声） | 35（高升） | 全降 | 对、半、四 |
| 阳去（六声） | 13（低升） | 全降 | 卖、树、用 |
| 阴入（七声） | 5（短促） | 无 | 急、黑、歇 |
| 阳入（八声） | 2（短促） | 无 | 白、读、药 |

**关键的"入声"现象**（上海话最显特色）：
- 上海话**保留完整的入声字**（以 -p/-t/-k 收尾的字），普通话已经没有了
- 例："吃"读"恰（qia）"、白"百"、"十"读"实（shíh）"、八"博（bō）"
- 入声**短促有力**，听着像"打嗝"
- **AI 在文字聊天里很难体现入声**（因为普通话已经没有入声字了），但要让 AI **知道**这个特点，避免用普通话入声字去套上海话

### 2.2 声母

1. **保留全浊声母**（最显著）：
   - 上海话保留中古汉语的"全浊声母"（b/d/g 等），普通话已经清化了
   - 例："白"读"ba"，不像普通话"bái"那样"清"——听着更"沉"
   - "地"读"di"，"坐"读"zo"
   - **这个特征是上海话最明显的"古音"**——听着比普通话"老派"

2. **n/l 自由**：
   - "你"读"侬（nóng）"，"来"读"来（le）"——基本分清
   - 但部分字 n/l 自由

3. **f/v 自由**：
   - "飞"读"fi"，"肥"读"vi"
   - 部分上海人 v 声母字较普通话多

4. **零声母字较多**：
   - 上海话里一些零声母字带轻微喉塞音，比如"安、爱、欧"开头有停顿

### 2.3 韵母

1. **保留鼻化音**（最显著）：
   - 上海话有完整的鼻化音（元音带鼻音），普通话基本没有
   - 例："搬"读"bǒ"（带鼻化）、"端"读"dǒ"、"关"读"guǒ"
   - **这个特征让上海话听着特别"软糯"**

2. **保留 -m 鼻音尾**：
   - "三"读"sē"（带 m 韵尾）、"甘"读"gē"、"心"读"xīn"
   - 普通话里 -m 已经并入 -n/-ng

3. **保留 -p/-t/-k 入声韵尾**（关键）：
   - "合"读"gep"、"十"读"zeh"、"白"读"baq"
   - **入声字的发音是上海话的"灵魂"**

4. **单元音多，复元音少**：
   - 上海话里"ai/ei/ou"等复元音较少，多数是单元音
   - "来"读"le"，"累"读"le"

### 2.4 节奏与腔调

上海人说话的节奏特点：

- **语速中等偏快**：上海人说话不慢，但也不像东北人那么冲
- **吐字清晰**：上海话"硬辅音"多，听着比普通话"清楚"
- **"嗲"味**：上海话（尤其是女性）有一种特殊的"嗲"味，发音位置靠前，听着"软"
- **"糯"**：上海话鼻化音多、入声短促，听着"糯"
- **"腔调"**：上海话的"腔调"（语气）比内容更重要——同样一句话，语气对了就"沪"，语气错了就"普"

## 四、语法特征

### 4.1 句法差异重点

1. **"伐"是万能否定词**：
   - "去伐？"= 去吗？
   - "好伐？"= 好不好？
   - "晓得伐？"= 知道吗？
   - **比普通话"不"用得更频繁**——上海话基本不用"不"来提问

2. **"伐要"代替"别/不要"**：
   - "伐要去"= 别去
   - "伐要忒"= 别太
   - "伐要客气"= 别客气

3. **"蛮" + 形容词**：
   - "蛮好"= 很好
   - "蛮大"= 很大
   - "蛮灵光"= 挺聪明
   - **比"很"更地道**

4. **"邪气" + 形容词**：
   - "邪气好"= 非常好
   - "邪气闹"= 非常热闹
   - **"邪气"是上海话的"超级"**

5. **"伐"作完成态标记**：
   - "吃过饭伐？"= 吃过饭了吗？
   - **用"伐"提问时，动词不变**（不像普通话要加"了"）

6. **"伊"作第三人称**：
   - "伊去"= 他去
   - "伊是阿拉爷"= 他是我爸爸
   - **保留古汉语用法**

7. **"覅"代替"不要"**：
   - "覅去"= 不要去
   - **比"伐要"更干脆**

8. **"邪气"作程度副词**：
   - "邪气好吃"= 非常好吃
   - "邪气大"= 非常大

9. **动词后置**（部分）：
   - "把伊吃脱"= 把它吃掉
   - "把伊弄好"= 把它弄好

10. **数量表达**：
    - "一只"= 一个
    - "一眼眼"= 一点点
    - "一眼眼眼"= 一点点点

### 4.2 节奏与口头禅

- **"侬晓得伐"** = 你知道吗（万能开场）
- **"伊讲"** = 他说（万能引语）
- **"阿拉讲"** = 我说
- **"伐要忒"** = 不要太
- **"邪气好"** = 非常好
- **"老好"** = 很好
- **"嘎讪胡"** = 闲聊
- **"茄山河"** = 闲聊
- **"没伊事体"** = 没关系
- **"勿来事"** = 不行
- **"蛮好"** = 很好
- **"顶好"** = 最好
- **"作孽"** = 可怜/造孽
- **"作"** = 矫情

### 4.3 句末语气词

- **"哦"** = 吧（句末）："侬好哦" = 你好吧
- **"伐"** = 吗："去伐" = 去吗
- **"了"** = 了："走了" = 走了
- **"咗"** = 了（粤式，吴语也用）："吃咗" = 吃了
- **"嗨"** = 啦："吃饭嗨" = 吃饭啦
- **"嗒"** = 啊："好嗒" = 好啊
- **"嗰"** = 嘛/呢："弄啥嗰" = 弄啥呢
- **"辣"** = 在/到："辣伊面" = 在他那边
- **"过"** = 过："去过" = 去过
- **"趟"** = 趟/次："一趟" = 一趟
- **"场"** = 场/次："一场" = 一场
- **"桩"** = 件/桩："一桩事体" = 一件事

## 五、对话句式（20+ 常用句式）

### 5.1 日常问候

1. **"侬好伐？"** = 你好吗？
2. **"阿拉好呀！"** = 我很好啊！
3. **"侬吃了伐？"** = 你吃了吗？
4. **"吃了呀，吃过生煎。"** = 吃了，吃过生煎。
5. **"侬伐要忒客气哦。"** = 你别太客气啊。
6. **"慢慢叫哦。"** = 慢慢来啊。
7. **"走啦！"** = 走了（道别）
8. **"明朝见哦！"** = 明天见啊！

## 六、书面表达方式

### 6.1 上海话在文字聊天中怎么"打出沪味"

**核心原则**：上海话的"沪味"靠**谐音字**+**句末语气**还原。

**具体技巧**：

1. **核心代词用谐音字**（最关键）：
   - "阿拉"代替"我/我们"
   - "侬"代替"你/您"
   - "伊"代替"他/她/它"
   - **这 3 个字是上海话的灵魂，用上立刻地道**

2. **"伐"代替"吗/不"**：
   - "晓得伐"= 知道吗
   - "好伐"= 好不好

3. **"伐要"代替"别/不要"**：
   - "伐要去"= 别去
   - "伐要忒"= 别太

4. **"蛮/邪气"代替"很/非常"**：
   - "蛮好"= 很好
   - "邪气好"= 非常好

5. **"晓得"代替"知道"**：
   - "侬晓得伐"= 你知道吗

6. **"事体"代替"事情"**：
   - "没伊事体"= 没关系

7. **"屋里厢"代替"家里"**：
   - "阿拉屋里厢"= 我家

8. **"小囡/囡囡"代替"女儿"**：
   - "阿拉囡囡"= 我女儿

9. **句末用"伐"提问**：
   - "侬好伐？"= 你好吗？
   - "吃过饭伐？"= 吃过饭了吗？

10. **句末用"哦"软化**：
    - "侬晓得哦"= 你知道啊
    - "慢慢叫哦"= 慢慢来啊

11. **善用"伐要忒"软化**：
    - "伐要忒客气"= 别太客气
    - "伐要忒急"= 别太急

12. **善用"作"**：
    - "作"= 矫情
    - "作孽"= 可怜
    - "作死了"= 矫情死了

### 6.2 文字聊天风格示例

**沪味浓的写法**：
> 侬好伐？阿拉屋里厢小囡发烧了，邪气急！侬晓得伐附近哪家医院好伐？伐要忒远哦。

**沪味淡的写法**：
> 你好么？我家女儿发烧了，非常急！你知道附近哪家医院好么？别太远啊。

**AI 模拟建议**：用"沪味浓的写法"，但不要每句话都这样。**关键代词（阿拉/侬/伊）+ 1-2 个沪味词 + 1 个"伐"提问**，就已经"沪味十足"了。

### 6.3 反面示例（AI 不要这么写）

- ❌ "侬好"用得太密（显得"装"）
- ❌ 把"上海话"硬翻译成"阿拉赞"（"阿拉"≠"我们"的简单替换）
- ❌ 用"小瘪三"、"戆度"等粗话（对外地人/陌生人慎用）
- ❌ 滥用"邪气"显得夸张
- ❌ 把"晓得"写成"晓特"（"晓得"标准写法）

## 八、民间习俗与里程碑事件表达

### 8.1 上海弄堂文化

**弄堂**（lòng táng）是上海的灵魂，相当于北京胡同：
- **弄堂口** = 弄堂入口，相当于北京"胡同口"
- **弄堂里厢** = 弄堂里面
- **亭子间** = 弄堂里的小阁楼（很小的房间）
- **灶披间** = 弄堂里的公用厨房
- **客堂间** = 弄堂里的公用客厅
- **晒台** = 弄堂里公用晒衣服的地方

**弄堂文化**：
- 上海弄堂里邻居关系亲密
- "张阿姨"、"李爷叔"是真亲
- 弄堂里嬢嬢们的"嘎讪胡"是日常
- 晾衣服、晒被子、孵太阳是弄堂一景
- 老虎灶（公共热水）已成历史

### 8.2 上海石库门

**石库门** = 上海特有建筑形式：
- "石库门"= 用石头箍住门框的弄堂房子
- 19 世纪末 20 世纪初上海开始建石库门
- 典型结构：前门 → 天井 → 客堂间 → 灶披间 → 后天井 → 后门
- 二楼：亭子间 + 厢房
- **石库门曾是上海中产阶级的家**——鲁迅、巴金都住过石库门

### 8.3 上海老洋房

**老洋房** = 上海老式花园洋房：
- 多在徐汇、静安、黄浦（原卢湾）
- 思南公馆、汾阳路、建国西路、武康路、湖南路是洋房区
- 老洋房有：花园、客厅、舞厅、仆人房
- **老克勒们的家**

### 8.4 沪剧与滑稽戏

**沪剧**：
- 上海本土戏曲，源于上海郊区田山歌
- 沪剧是上海"唯一"的本土戏曲
- 早期沪剧用上海话唱
- **代表剧目**：《罗汉钱》、《芦荡火种》（京剧《沙家浜》前身）、《星星之火》

**滑稽戏**：
- 上海本土曲艺
- 独角戏、双人戏、多人戏
- 代表人物：周柏春、姚慕双、杨华生、王汝刚
- **用上海话表演，笑话讽刺**
- 滑稽戏是上海话最"活"的舞台

### 8.5 上海咖啡文化

上海是中国咖啡馆最多的城市：
- 19 世纪咖啡馆传入上海
- 老大昌、凯司令是上海老牌咖啡馆
- **上海人爱喝咖啡**——早饭咖啡、下午茶咖啡、饭后咖啡
- "小布尔乔亚"情调 = 咖啡 + 文学 + 电影
- **武康路、安福路、巨鹿路** 是上海咖啡馆密度最高的地方

### 8.6 旗袍文化

**旗袍** = 上海的标志之一：
- 20 年代上海旗袍兴起
- 海派旗袍 = 海派文化代表
- 张爱玲是旗袍的标志人物
- **旗袍是上海"小布尔乔亚"文化的象征**
- 现在的上海旗袍店：蔓楼兰、瀚艺、旗袍会

### 8.7 上海本帮菜

**本帮菜** = 上海本地菜：
- **浓油赤酱** = 上海菜特点
- 代表菜：红烧肉、油爆虾、糖醋小排、腌笃鲜、咸肉菜饭、八宝鸭、油酱毛蟹
- **本帮菜特点**：偏甜、偏红、偏浓
- 老饭店、德兴馆、兰心餐厅是本帮菜名店

### 8.8 老克勒文化

**老克勒**（Old Class / Old School）：
- 1840-1949 年代的上海外国人带来的"洋派"生活方式
- **老克勒特征**：
  - 穿西装、吃西餐、喝咖啡
  - 讲英文、懂规矩
  - 跳舞、打网球、看赛马
  - 家里有佣人
- **现代"新克勒"**：受过良好教育、有国际视野、讲究生活品质的上海人

### 9.3 常见错误

- ❌ "侬"用得太密（显得"装"）

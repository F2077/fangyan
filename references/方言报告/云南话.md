# 云南话研究手册

> 面向「Claude Code 方言对话插件」的素材文档
> 目标：让 AI 在文字对话中能"用地道云南话跟云南用户聊天"

---

## 1. 概述

云南话，准确说叫**云南汉语方言**，在方言学上属于**汉语-西南官话-昆贵片-滇中片**，以**昆明话**为代表点。云南是中国民族最多的省份（26 个世居民族），汉语方言与少数民族语言（彝、傣、白、哈尼、纳西、傈僳、景颇、普米等）长期接触融合，形成一种"听起来像四川话但慢半拍、软一截、古朴又鲜活"的独特味道——如果说成都话是盖碗茶里泡着的茶叶，昆明话就是**翠湖边抽水烟筒的大爷、慢悠悠一句话能拖三拍**的闲适；如果说重庆话是码头江湖气，云南话就是**高原阳光底下、菌子锅里咕嘟着的烟火气**。

云南话的三大底色：**第一慢**，语速全国倒数，说话尾音拖长，像昆明四月的风；**第二古**，保留不少古汉语词，"得是""恁个""咋个"都是活化石；**第三野**，受少数民族语影响，用词生猛活泼，"整""搞""冲""猛"一股脑上场。云南人说话的核心节奏是——**先拖个"呢"、再垫个"噶"、最后收个"嘛"**，一句话听起来像在唱小调，不急不赶。

记住一个核心比喻就够了：**云南话是坐在西山龙门上看滇池**——慢、远、亮、带点醉意。

## 2. 语音特征

### 2.1 声调（4 声调系统，无入声）

云南话与川渝同属西南官话，**入派阳平**，古入声字归入阳平。

| 古音 | 例字 | 云南话 | 普通话 |
|---|---|---|---|
| 入声 | 八、吃、七、出、桌 | 阳平 | 阴平 |
| 入声 | 急、笔、屋、哭 | 阳平 | 阴平 |
| 入声 | 直、白、读、毒 | 阳平 | 阳平（巧合） |

**四声调值（昆明话参考）**：
- 阴平 44（半高平）：天、三、妈
- 阳平 31（低降调）：人、白、棉
- 上声 53（高降调）：我、纸、走
- 去声 213（降升调）：是、对、饭

**昆明话特有的调值特点是"降得柔、升得缓"**，比成都话更软，比重庆话更拖，听感上偏"慵懒"。

### 2.2 声母特点

1. **n-l 不分**："南"="蓝"，"男"="兰"，"脑"="老"。这是云贵川共通点。
2. **f-h 不分（昆明片有部分保留）**："灰"读作"飞"，"荒"读作"方"，但比成都话保留得多些。
3. **平翘舌不分**：zh/ch/sh 与 z/c/s 自由切换。"吃饭"≈"七凡"，"知道"≈"几道"。
4. **r 声母读作 [l] 或零声母**：把"人"读作近似"len"，把"热"读作近似"le"。
5. **零声母保留**：古疑母字"我、饿、矮、安"保留 ng- 声母。
6. **玉溪、建水片保留古精组声母**："酒"="走"、"秋"="抽"，比昆明更古。
7. **昭通话（属西南方言滇东北片）入声保留较多**，与四川话接近，但带云南味。

### 2.3 韵母特点

1. **前后鼻音不分**：en/in 与 eng/ing 自由切换，"陈"="程"，"林"="灵"，"心"="星"。
2. **-u 韵尾保留**：古模韵字"路、读、图"在云南话里保留 [-u] 韵尾。
3. **儿化几乎不用**：云南话不儿化，但有独特的"子"尾，"娃子""馆子""勺子"。
4. **"啊"作语气词时的特殊变调**：句末的"啊"常读成轻声"呀"，带拖音。"好吃噶"——"噶"会拖成"噶啊——"。
5. **"哦"读作 [ɔ]**：比普通话更"圆"，"我"读作"握"。

### 2.4 语流特点（最具识别度的"滇味"）

1. **拖长音慢节奏**：陈述句尾下拉，节奏比成都话更慢。"要得"读作 "yao der"，尾音拖一拍。
2. **句末语气词必有"呢""噶"**：每句话结束都得"呢——"或"噶——"拉长。
3. **轻声使用普遍**：虚词几乎全部轻声。
4. **连续变调频繁**：三字、四字短语里出现连续上声变调。

### 2.5 各地口音速辨

- **昆明话**：最标准、最慢、最"懒洋洋"，尾音拖长。
- **玉溪话**：比昆明略硬，保留部分古音。
- **建水话**：古城音重，保留入声痕迹，文人气。
- **昭通话**：偏四川话，语速快，山区气。
- **大理话（白语底层）**：夹杂白语词，"阿鹏""金花"常出现。
- **丽江话（纳西底层）**：带纳西语色彩，叠词多。

## 4. 语法特征

### 4.1 句末语气词系统（云南话的灵魂）

云南话的句末语气词系统比川渝更复杂，几乎每句话都得带一个：

- **噶（嘎）**：万能尾音，"好吃噶？""走噶？"
- **呢**：拖长，"吃呢？""走呢？"
- **嘛**：撒娇、商量，"吃嘛""走嘛"
- **咯**：完成，"吃咯""走咯"
- **噻**：催促、邀请，"走噻""吃噻"
- **呢咯**：组合，更拖，"你走呢咯？"
- **呢嘛**：组合，撒娇，"好吃呢嘛"
- **噶嘛**：组合，"吃噶嘛"
- **嘛噶**：组合，"吃嘛噶"

**组合规律**：陈述句尾用"呢""咯""噶"；疑问句尾用"噶""噻"；命令/邀请句尾用"嘛""噻"；撒娇用"嘛""呢嘛"。

### 4.2 "给""得"作语气副词

- **给是 / 得是** + 句子：表反问、猜测
  - "给是你没吃饭？" = 是不是你没吃饭？
  - "给有下雨？" = 有没有下雨？
  - "得是嫩个？" = 是这样吗？

### 4.3 "整"作万能动词

云南话"整"比成都话还万能，几乎所有动词都可以替换：

- **整饭**：做饭 / 吃饭
- **整酒**：喝酒
- **整两杯**：喝两杯酒
- **整一哈**：弄一下
- **整不成**：办不到 / 不行
- **整得成**：办得到 / 可以
- **整日气**：找气受
- **整人**：捉弄人
- **整点**：弄一点

### 4.4 双宾语/处置式特点

- **把字句** 与普通话类似："把这个整掉"
- **给字句**："给我走" = 让我走 / 快走（命令式，云南话特有命令语气）

### 4.5 "挨"作介词

古汉语遗留，"挨" = 跟、和：

- "挨你说" = 跟你说
- "挨他讲" = 跟他讲
- "挨你借本书" = 跟你借本书

### 4.6 "莫"作否定前缀

- **莫得**：没有
- **莫去**：别去
- **莫搞**：别搞
- **莫整**：别弄
- **莫怕**：别怕

### 4.7 叠词/重复表程度

- "慢慢走""好好吃""高高兴兴"
- "黑黑""亮亮""香香""甜甜"
- "嫩个""恁个""咋个"作程度副词

### 4.8 民族语底层影响

云南话夹杂少数民族语词汇：
- "阿鹏""金花"（白族）
- "琵琶肉"（傈僳族）
- "烤乳猪"（傣族）
- "苦荞"（彝族）
- "乳扇"（白族）

## 6. 书面表达方式（怎么"打出滇味"）

### 6.1 必带语气词

每句话末尾必须拖一个语气词，否则就失去滇味：

- "我走呢。" 不如 "我走呢嘛。"
- "好吃。" 不如 "好吃噶嘛。"
- "你走不走？" 不如 "你走不走噶？"

### 6.2 语气词连用

- "呢嘛""噶嘛""呢咯""嘛噶"
- 一句话末尾可叠加："好吃呢嘛噶。"

### 6.3 用 "给" 不用 "是"

- 普通话："是不是？"
- 云南话："给是？""给呢？"

### 6.4 用 "嫩个""恁个" 表程度

- 普通话："这么多""这么贵"
- 云南话："嫩个多""恁个贵"

### 6.5 用 "整" 作万能动词

- "整饭吃""整一哈""整不成"

### 6.6 用 "莫" 表否定

- "莫去""莫得""莫搞"

### 6.7 节奏拖长

- "好吃噶——" "走呢——" "嫩个贵呢——"
- 用破折号或省略号模拟拖音

### 6.8 夹杂食物/地名

- "走，甩碗米线噶——"
- "上翠湖呢嘛。"
- "整点饵块吃嘛。"

### 6.9 避用"川式"语气词

- 不要过度用"哦""哈""噻"（成都话标志）
- 多用"噶""呢""嘛""咯"

### 6.10 善用食物比喻

- "这个人咋个嫩个瓜嘛，跟见手青中毒呢一样。"
- "嫩个贵呢，抢人噶。"

## 9. 给 AI 的使用建议

### 9.1 慢、拖、长

云南话的核心是"慢"，AI 回复每句话都要带拖音感："好吃噶——""走呢嘛——"。比川渝话还要慢半拍。

### 9.2 句末必带语气词

**每句话末尾必须拖一个语气词**："噶""呢""嘛""咯""噻"，否则就失去滇味。一句话可叠加："好吃呢嘛噶——"。

### 9.3 用"给""得"不用"是不是"

疑问/反问必用"给是""得是""给呢""得呢"，不用"是不是"。"给是你？""得是嫩个？"

### 9.4 万能动词"整"

把"做""弄""搞""吃""喝""打""骂"全部用"整"代替："整饭""整酒""整不成""整得成"。

### 9.5 用"莫"不用"别"

否定词用"莫"："莫去""莫得""莫搞"，比"别""没"更云南味。

### 9.6 用"嫩个""恁个"表程度

"这么""那么"用"嫩个""恁个"代替。"嫩个贵呢嘛""恁个多呢嘛"。

### 9.7 善用"呢喃""板扎""鬼火绿"

云南特色形容词必须用上："板扎得很嘛"（厉害）、"鬼火绿"（生气）、"呢喃"（娇气）。

### 9.8 善用食物/地名

云南人聊天离不开食物和地名："走，甩碗米线噶——""上翠湖呢嘛——""整点烧饵块呢嘛——""普洱茶呢嘛，板扎得很嘛——"。

### 9.9 菌子文化必带

夏天菌子话题是云南特色，AI 应主动提及："菌子上市呢嘛，见手青嫩个板扎呢嘛。"

### 9.10 慢节奏 + 阳光感 + 烟火气

最后一条最关键：云南话的灵魂是"慢节奏 + 阳光感 + 烟火气"。翠湖边大爷抽水烟筒、米线馆嬢嬢喊你吃米线、抚仙湖渔民收网回家——AI 回复时把这种"高原阳光底下的烟火气"传达出来，每句话都像在晒太阳、像在甩米线、像在喝茶。**快嘴、利落、北方式的对话，绝对不是云南味。**

# 山东话方言研究

> 面向 Claude Code 方言对话插件，给程序员和产品经理看的"地道山东话"指南。目标：让 AI 在文字聊天里像济南老师儿、青岛小哥、烟台渔民、潍坊老乡、济宁汉子、菏泽爷们儿那样跟山东用户拉呱、喝酒、吹牛。

---

## 1. 概述

山东话，是中国山东省使用的汉语方言，属北方方言区，但内部差异极大。**狭义山东话**指山东中部、西北部、西南部说的**中原官话**（济南话、济宁话、聊城话、菏泽话、临沂话西部）；**胶辽官话**（青岛话、烟台话、威海话、潍坊东部）则属于另一个方言区，特点更接近东北话和大连话；**冀鲁官话**分布在德州、滨州一带。

山东话最显眼的几条特征：

- **直爽豪迈**：说话不绕弯，"是就是，不是就不是"，江湖气重；
- **爱称"老师儿"**：见面喊"老师儿"，不是尊称而是日常称呼，男的也喊，女的也喊，外地人听着像叫老师；
- **酒桌文化全国第一**：劝酒词一套套的，"感情深，一口闷"；
- **保留入声字**：尤其是济南话里"吃、白、客、北、桌"等古入声字还读短促调；
- **语气词丰富**：么、咧、嘞、哈、呀、呗、昂、哎、嗨轮；
- **副词"贼"**：修饰形容词表示"非常"，"贼香、贼好、贼得劲"；
- **副词"杠/杠赛来/杠好"**：表示"非常、特别"；
- **爱说"得（děi）"**：表示"舒服、好"或"需要"；
- **称呼爱用儿化**：老师儿、媳妇儿、闺女儿、兄弟儿、姊妹儿。

山东话的形成跟**齐鲁文化**密不可分。孔孟之乡、礼仪之邦，山东人说话讲究"义气""厚道"，再加上闯关东走西口带出去的山东话，被全国人民熟悉。山东快书、山东梆子、吕剧、柳琴戏、茂腔这些曲艺又把山东话推向了全国。

让 AI 学山东话，学的不是"把'什么'换成'啥'"，而是**"怎么说话像山东爷们儿"——硬气、敞亮、实诚、不酸不唧、不装洋蒜**。

## 2. 语音特征

> 文字聊天里这些音大多靠谐音字和语气词来传达。下面是给 AI 听音频时的参照。

### 2.1 声调

山东话声调最显著的特点：**济南话保留三个入声调**（阴入、阳入、次浊入），古入声字读短促调，调值约为 **4**（阴入）、**23**（阳入），例如：

- **阴入**：吃 cī、八 bā、桌 zhuō、一 yī、七 qī、接 jiē、歇 xiē；
- **阳入**：白 bái（短）、直 zhí（短）、十 shí（短）、宅 zhái（短）、读 dú（短）；
- **次浊入**：月 yuè（短）、肉 ròu（短）、六 liù（短）、麦 mài（短）、绿 lǜ（短）、白 bái（短）。

**整体调值**：
- 阴平 **213**（低降升）；
- 阳平 **42**（中降）；
- 上声 **55**（高平，不像普通话的 214 拐弯）；
- 去声 **31**（低降）。

听起来**调子平、节奏硬、字头咬得重**，跟河南话接近但比河南话更冲。

### 2.2 声母

- **平翘舌不分**：济南、潍坊、济宁等地，zh、ch、sh 常读成 z、c、s。"知道"→"zi dao"、"吃"→"ci"、"山"→"san"。**青岛话不混**，青岛话反而平翘分明（受胶辽官话影响）。
- **r 读如 l 或 y**：济宁、菏泽一带"人、热、肉"常读如"lén、lè、lòu"；青岛烟台话"r"读成"y"或"l"。
- **n/l 不分**：鲁西南更明显，"牛奶"读"流奶"。
- **日母字特殊**："让、如、热"在胶东话里读 y- 声母或零声母，"热"读如"yè"。
- **f/h 互换**：少数字"飞、灰"在鲁西读如"灰、飞"。

### 2.3 韵母

- **儿化极重**："老师儿、媳妇儿、闺女儿、兄弟儿、姊妹儿、小孩儿"。
- **前后鼻音不分**：济南话里 -n/-ng 收尾常合流，"陈、根、跟"都读 -ng 收尾。
- **o/e 混读**："我"读 wǎ 或 ě，"可"读 kě。
- **ai/ei 互换**：胶东话"白、百"读如"bē"。
- **合音字多**："这么"→"啧"、"那样"→"那样"→"nèng"、"知道"→"zháo"。

### 2.4 整体腔调

山东话听着**硬、冲、有劲**：

- 调子偏平，拐弯少，节奏明快；
- 字头咬得重，字腹松，句尾常往下砸；
- 嗓门大，爱拔高；
- 句中爱停顿、加"咧、哈、嘛、昂"；
- 整体像在**喊山、唱号子**，豪迈感足。

## 4. 语法特征

### 4.1 副词"贼"修饰形容词

山东话最具标志性的语法点。**"贼"作为程度副词，等于"非常"，可修饰形容词和心理动词**。

- "这酒贼好！" = 这酒非常好！
- "煎饼卷大葱贼香！" = 煎饼卷大葱非常香！
- "这酒喝得贼得劲！" = 这酒喝着非常舒服！
- "这家店服务贼差！" = 这家店服务非常差！
- "这事儿贼邪乎！" = 这事儿非常邪门！

"贼"是**万能程度副词**，比"很"更地道，更山东味。"挺、怪"也用，但不如"贼"过瘾。

### 4.2 AA 式重叠

山东话里大量使用**AA 式重叠**，表示"稍微、短暂"或加强语气：

- "你等等" → "你等儿等儿"
- "看看" → "看看儿"
- "尝尝" → "尝尝儿"
- "说说" → "说说儿"
- "想想" → "想想儿"
- "走走" → "走走儿"
- "溜溜" → "溜溜儿"

也可加儿化。"AA 儿"表示动作短暂、轻松，"尝尝儿"就是"尝一点"。

### 4.3 被动句与"叫"字句

山东话常用**"叫"字被动句**，"叫"等于普通话的"被、让"：

- "叫老师剋了一顿" = 被老师批评了一顿
- "叫他揍了" = 被他打了
- "叫雨淋了" = 被雨淋了
- "叫风吹跑了" = 被风吹跑了
- "叫狗咬了" = 被狗咬了

鲁西南也用"让"："让他打了"。"被"字句在山东话里也用，但不如"叫"字句地道。

### 4.4 "得"字句与程度补语

山东话**"得"字句特别发达**，补语在"得"后：

- "好得很" = 好得很
- "好得了" = 好了
- "香得很" = 香得很
- "香得了" = 香了
- "饿得慌" = 饿得不行
- "冷得慌" = 冷得不行
- "累得慌" = 累得不行
- "忙得慌" = 忙得不行
- "困得慌" = 困得不行
- "渴得慌" = 渴得不行
- "疼得慌" = 疼得不行
- "高兴得了" = 高兴得不行
- "累得了" = 累得不行

**"得慌"** 是山东话特有结构，强调程度之深。

### 4.5 "得（děi）"的多重含义

山东话里"得（děi）"特别多义：

1. **舒服、得劲**："睡得（děi）很" = 睡得舒服得很
2. **需要**："你得（děi）喝点" = 你需要喝点
3. **行、可以**："得（děi）行" = 行
4. **差不多**："得（děi）乎" = 差不多
5. **表示程度深**："好得（děi）很" = 好得很

### 4.6 动词后缀

山东话动词后缀丰富：

- **"巴"**：试试儿巴 = 试试看
- **"咕"**：摆弄咕 = 摆弄
- **"达"**：摔达 = 摔
- **"拉"**：扒拉 = 扒拉
- **"腾"**：折腾 = 折腾
- **"乎"**：忙乎 = 忙活
- **"活"**：忙活 = 忙
- **"道"**：说道 = 说
- **"咧"**：走咧 = 走了

### 4.7 后缀"子"和"儿"

- **"子"后缀**：桌子、椅子、盖子、盆子、罐子、瓶子、筷子、勺子、叉子、刀子、剪子、锤子、凿子、钻子、钩子、套子、被子、褥子、毯子、褥子。
- **"儿"后缀**：老师儿、媳妇儿、闺女儿、兄弟儿、姊妹儿、小孩儿、小子、妹子、小子儿、鸟儿、鱼儿、花儿、叶儿、瓜儿、果儿、馅儿、味儿、事儿、味儿、劲儿、门儿、牌儿、车儿、房儿。
- **不加后缀**：也大量存在，比如"馒头、煎饼"。

### 4.8 "么"和"啥"的选择

山东话里**"么"（mè）**比"啥"更地道，尤其在济南、济宁、菏泽：

- "你干么？"（你干什么？）
- "你说么？"（你说什么？）
- "有么事？"（有什么事？）

**"啥"** 在胶东、鲁北也常用，但胶东人更爱说"么个"或"哈"。

### 4.9 "咋"和"怎么"

**"咋"** = 怎么，最常用：

- "咋了？" = 怎么了？
- "咋办？" = 怎么办？
- "咋说？" = 怎么说？
- "咋弄？" = 怎么弄？

**"怎么"** 也用，但更书面。**"咋"** 更口语、更山东。

### 4.10 比较句

山东话比较句式：

- "我比你高" → "我比你高"
- "我比恁高" → "我比你高"（恁 = 你）
- "A 比 B 强" → 通用
- "A 不如 B" → 通用
- "A 没 B 好" → "A 不如 B 好"
- "A 跟 B 一样" → "A 跟 B 一样"

最高级表达：

- "杠好" = 特别好
- "最最好的" = 最最好的
- "最最最" = 最最最（多重叠加强调）

### 4.11 完成体"了"

山东话"了"用得跟普通话类似，但**常与"罢"结合**：

- "吃了没？" → "吃罢了没？" = 吃了吗？
- "走了" → "走罢" = 走了
- "干完了" → "干罢了" = 干完了

**"罢"** 是山东话里表完成的特殊虚词。

### 4.12 进行体"着"和"咧"

- "吃着" → "吃咧"（鲁西南）
- "干着" → "干咧"（鲁西南）
- "看着" → "看咧"

**"咧"** 也用于进行体。

### 4.13 介词

- "搁" = 在、到。"搁这儿" = 在这儿
- "打" = 从。"打济南来" = 从济南来
- "赶" = 到、往。"赶明儿" = 到明天
- "连" = 连。"连他都不知道" = 连他都不知道

### 4.14 连词

- "要不" = 要么。"要不你甭去了"
- "要不的话" = 要不的话
- "要不咱这样" = 要不咱这样
- "要么" = 要么。"要么你去，要么我去"
- "要不" = 要不。"要不我去？"
- "搁要" = 如果。"搁要你来，吱一声"
- "要是" = 要是。"要是你来，就好了"
- "如要" = 如果
- "即使" = 即使
- "虽然" = 虽然
- "不管" = 不管
- "甭管" = 不管。"甭管他"

## 6. 书面表达方式

山东话在文字聊天里怎么打出鲁味？

### 6.1 儿化字必须带"儿"

- "老师儿"（不写"老师"）
- "媳妇儿"（不写"媳妇"）
- "闺女儿"（不写"闺女"）
- "兄弟儿"（不写"兄弟"）
- "姊妹儿"（不写"姊妹"）
- "小孩儿"（不写"小孩"）
- "哥们儿"（不写"哥们"）
- "味儿"（不写"味"）

### 6.2 副词必带

- "贼好"（不用"很好"）
- "杠好"（不用"非常好"）
- "杠赛来"（必须带"来"）
- "忒好"（不用"太好"）
- "怪好"（不用"挺好"）
- "真恣儿"（不用"真舒服"）

### 6.3 代词

- "俺"（不用"我"，至少不能全文都用"我"）
- "咱"（不用"我们"，山东话里"咱"更亲切）
- "恁"（不用"你"，写出来更有山东味）
- "么"（不用"什么"）
- "啥"（也可用，但"么"更正）
- "咋"（不用"怎么"）
- "哈"（不用"什么"加重）
- "嘎哈"（不用"干什么"）
- "多咱"（不用"什么时候"）
- "一崩儿"（不用"一会儿"）

### 6.4 语气词必加

- "咧"（不用"呢"或"啊"）
- "嘞"（不用"了"）
- "哈"（语气加重）
- "呗"（表"罢了"）
- "昂"（表"啊"）
- "嘛"（表"啊"）

### 6.5 短句、感叹句、语气词堆叠

- "中！"（不用"行"或"好"）
- "不中！"（不用"不行"）
- "杠好！"（不用"非常好"）
- "贼香！"（不用"特别香"）
- "得劲！"（不用"舒服"）
- "恣儿！"（不用"痛快"）
- "美得很！"（不用"很高兴"）

### 6.6 谐音字

- "么" = 什么
- "咋" = 怎么
- "恁" = 你
- "俺" = 我
- "咱" = 我们
- "咧" = 呢/啊
- "嘞" = 了/呢
- "昂" = 啊/嗯
- "呗" = 罢了
- "呗呗" = 罢了罢了
- "拉倒" = 算了
- "哈" = 喝（"哈啤酒"）
- "恣" = 舒服

### 6.7 不打"的"，用"嘞"

- "我去嘞" = 我去了
- "他来咧" = 他来了
- "吃罢咧" = 吃完了

### 6.8 "得"字的特殊用法

- "睡得（děi）很" = 睡得很舒服
- "这酒得（děi）喝点" = 这酒要喝点
- "你得（děi）行" = 你还行
- "这事得（děi）乎" = 这事差不多

### 6.9 拼音标注

为了解音，必要时附拼音：

- 么（mè）
- 恁（nèn）
- 俺（ǎn）
- 嘞（lei）
- 咧（lie）
- 恣（zì）
- 得（děi）
- 捯饬（dáo chi）
- 毛嗑（máo kè）
- 杠赛来（gàng sài lái）
- 嘎哈（gà ha）
- 干哈（gàn há）

### 6.10 标点、emoji

山东话文字聊天标点用法：

- **！连用**：表示激动、肯定。"杠好！！"
- **？连用**：表示怀疑。"你说啥？？？"
- **……连用**：表示无奈、思考。"这样啊……"
- **哈哈哈**：爽朗的笑，山东爷们儿的笑点比较低。"哈哈哈哈，杠好！"
- **哈哈哈** 不用 **嘿嘿嘿**：嘿嘿更南方，山东爷们儿不这么笑
- **得瑟** 表调侃常用 😂 或 😏

emoji 不用太多，山东爷们儿不爱发 emoji，顶多：

- 🍺🍻（酒桌）
- 🌶️（吃辣）
- 🍻（干杯）
- 😂😏（调侃）
- 👍（认可）
- 👏（赞赏）
- 🙏（拜托）
- ❤️（关心）

## 7. 典型对话示例

### 示例 1：济南大明湖畔

```
A：老师儿，你搁哪儿咧？
B：俺在大明湖这边溜达咧！你来么？
A：中！俺马上过去！咱搁哪儿碰头？
B：搁大明湖西南门那儿，那个老槐树下头。
A：中中中，俺叫上俺兄弟儿一块儿。
B：好，你快着点儿咧！俺都等得（děi）慌了！
A：放心！一崩儿就到！
B：路上慢着点儿，别紧！
A：知道咧！马上到！

——普通话版——
A：师傅，你在哪儿呢？
B：我在大明湖这边散步呢！你来吗？
A：行！我马上过去！咱在哪儿碰头？
B：在大明湖西南门那儿，那个老槐树下面。
A：行行行，我叫上我哥们儿一块儿。
B：好，你快点儿啊！我都等得着急了！
A：放心！一会儿就到！
B：路上慢点儿，别着急！
A：知道了！马上到！
```

```
A：老师儿，青岛这海真得劲咧！
B：可不是嘛，栈桥这儿杠赛来！海风贼爽！
A：这海风吹着得（děi）很！你闻闻，有咸味儿。
B：哈哈哈，青岛这味儿就是咸鱼味儿！
A：来，咱哈瓶青啤！边哈边看海。
B：中！老板儿，来两瓶青啤！
A：青岛啤酒，杠赛来！
B：可不就是！青啤哈啤酒，恣儿得很咧！
A：再哈一杯！
B：走一个！
A：干哈！干了！
B：哈哈哈哈，得劲！

——普通话版——
A：师傅，青岛这海真舒服啊！
B：可不是嘛，栈桥这儿特别好！海风特别爽！
A：这海风吹着真舒服！你闻闻，有咸味。
B：哈哈哈，青岛这味就是咸鱼味！
A：来，咱喝瓶青岛啤酒！边喝边看海。
B：行！老板，来两瓶青岛啤酒！
A：青岛啤酒，特别棒！
B：可不就是！青岛啤酒喝啤酒，痛快得不行啊！
A：再喝一杯！
B：干一个！
A：干了！干了！
B：哈哈哈，舒服！
```

## 8. 民间习俗与里程碑事件表达

### 8.1 齐鲁文化与孔子、孟子

山东是**孔孟之乡**，曲阜有孔府、孔庙、孔林（"三孔"），邹城有孟府、孟庙。山东人说话常引用孔子的话：

- "孔子说了，'有朋自远方来，不亦乐乎！'"
- "孔子家乡人，得讲礼数咧！"
- "孟母三迁，咱得为孩子考虑！"
- "俺山东人，实在！实在！"

山东人常说"俺是孔子老乡"，透着骄傲。

### 8.2 泰山

泰山是**五岳之首**，山东话里说"登泰山" = "爬泰山"，跟山东人聊天他们会推荐你登泰山：

- "来山东，不登泰山，白来咧！"
- "登泰山，看日出，杠赛来！"
- "泰山石敢当，辟邪！"
- "泰山奶奶保佑！"

山东家里、门口、车上常挂"泰山石敢当"石雕，辟邪保平安。

### 8.3 趵突泉

济南**趵突泉**是"天下第一泉"，济南也叫"泉城"：

- "济南泉水多，72名泉咧！"
- "趵突泉的泉水，贼甜！"
- "到济南，不看趵突泉，白来咧！"
- "济南的甜沫，你喝过么？"
- "大明湖的荷花，杠赛来！"

### 8.4 青岛啤酒节

青岛国际啤酒节是**亚洲最大的啤酒节**，每年 8 月在青岛举办：

- "青岛啤酒节，哈啤酒，恣儿！"
- "青岛啤酒，杠赛来！"
- "来青岛不哈青啤，白来咧！"
- "青啤哈啤酒，山东爷们儿的标配！"

### 8.5 烟台国际葡萄酒节

烟台是**亚洲唯一的国际葡萄与葡萄酒城**：

- "烟台葡萄酒，杠赛来！"
- "烟台苹果莱阳梨，杠赛来！"
- "到烟台，得哈葡萄酒！"
- "张裕葡萄酒，那是国货之光！"

### 8.6 潍坊国际风筝节

潍坊是**世界风筝都**，每年 4 月举办国际风筝节：

- "潍坊风筝节，杠热闹！"
- "潍坊风筝，飘到全世界！"

### 8.7 菏泽国际牡丹节

- **接亲**：新郎带车队去新娘家接亲，过关斩将
- **回门**：婚后第三天，新娘新郎回娘家，叫"回门"
- **改口**：婚后新娘改口叫男方爸妈"爸、妈"，男方爸妈给红包
- **三拜**：一拜天地、二拜高堂、夫妻对拜

### 8.25 山东酒桌文化（重点强调）

**山东酒桌文化全国第一**，规矩多，讲究多：

1. **座次有讲究**：圆桌正对门的位置是主位，是长辈或主宾的座位；主位左右是次主位，按辈分、年龄排列。
2. **敬酒有讲究**：
   - 晚辈敬长辈：双手持杯，微微躬身；
   - 主人敬客人：第一杯要先敬主宾；
   - 客人回敬主人：表示感谢；
   - 同辈之间：互相敬酒，碰杯要低；
   - 主人提三杯酒：第一杯敬主宾，第二杯敬陪客，第三杯敬所有人。
3. **劝酒有套词**：
   - "感情深，一口闷！"
   - "感情铁，喝出血！"
   - "宁可胃上烂个洞，不让感情裂道缝！"
   - "你不喝就是看不起我！"
   - "男人不喝酒，白在世上走！"
   - "来来来，再走一个！"
   - "喝！喝！喝！"
4. **干杯方式**：
   - **端了**：干了
   - **走一个**：喝一个
   - **一口闷**：一口喝完
   - **满上**：倒满
5. **挡酒**：
   - "我真不能喝了！"
   - "我喝多了！"
   - "我喝点茶！"
   - "我挡挡酒！"
6. **代酒**：
   - "我替你喝！"
   - "我代你一杯！"
7. **下酒菜**：
   - 把子肉、九转大肠、油爆双脆、糖醋鲤鱼、四喜丸子、葱烧海参、拔丝地瓜、拔丝山药、油炸蚕蛹。
8. **酒后送人**：
   - 一定送回家
   - "老师儿，你喝多了，俺送你！"
   - "甭送，俺能走！"
   - "甭犟，必须送！"

### 8.26 山东饭桌礼仪

- **主位**：圆桌正对门的位置
- **长辈先动筷**：长辈先动筷，晚辈才能动筷
- **敬酒**：晚辈敬长辈要站起来，双手持杯
- **夹菜**：给长辈夹菜要用公筷
- **不浪费**：碗里不能剩饭
- **吃饭不吧唧嘴**
- **饭前洗手**：饭前要洗手
- **饭后打包**：吃不完打包，不浪费

## 9. 给 AI 的使用建议

> 这是给 AI 在文字聊天里用地道山东话跟山东用户聊天的实操建议。

### 建议 1：称呼直接用"老师儿"，别犹豫

山东话里**"老师儿"是万能称呼**，见谁都叫老师儿，男的女的老的少的都行。AI 在跟山东用户聊天时，称呼就用"老师儿"，显得亲切、接地气。比"您好""先生""朋友"都更山东味。

### 建议 2：副词"贼"和"杠"必须用

山东话里**"贼"和"杠"是灵魂副词**，能修饰大部分形容词：

- "这事儿贼好！" = 这事非常好！
- "这酒杠香！" = 这酒特别香！
- "这个老师儿贼实在！" = 这师傅特别实在！

AI 在夸赞、感叹时优先用这两个副词，立刻有山东味。

### 建议 3：语气词不能省

山东话里**语气词是灵魂**，"咧、嘞、哈、呗、昂"等用得多才像山东人：

- "你吃么咧？"（你吃什么呢？）
- "中嘞！"（行！）
- "咋哈？"（怎么呢？）
- "就这样呗！"（就这样吧！）
- "行昂？"（行吗？）

AI 不能省掉这些语气词，否则听起来像在播新闻联播。

### 建议 4：代词用"俺""咱""恁"

- "俺" = 我（鲁中、鲁西）
- "咱" = 我们
- "恁" = 你（鲁西南、鲁西）

不要全用"我"和"你"，**山东爷们儿爱用"俺""咱""恁"**。AI 在对话里混着用，立刻有山东味。

### 建议 5：入声字用短促调

山东话保留**入声字**，AI 在听音频时可以听出短促感。文字聊天里入声字要用对，比如：

- "白" = bái（短）
- "吃" = cī（短）
- "桌" = zhuō（短）
- "北" = bě（短）

**入声字用错就不像山东话**。AI 在文字里写"吃饭、喝酒、桌子、北方"这些词时，注意读音要短促。

### 建议 6：儿化字一定要写出来

**山东话儿化字多**，文字聊天里**必须打"儿"字**：

- 老师儿（不是老师）
- 媳妇儿（不是媳妇）
- 闺女儿（不是闺女）
- 兄弟儿（不是兄弟）
- 姊妹儿（不是姊妹）
- 小孩儿（不是小孩）

不打"儿"字就丢了山东味。

### 建议 7：酒桌文化单独强调

**山东酒桌文化全国第一**，AI 跟山东用户聊天如果涉及酒桌，必须：

- 用山东劝酒词（"感情深，一口闷"）
- 知道山东酒桌座次
- 知道山东酒桌规矩（长辈先动、敬酒站起、双手持杯）
- 知道山东人劝酒话术

### 建议 8：齐鲁文化要懂

山东是**孔孟之乡、礼仪之邦**，AI 跟山东用户聊天要懂：

- 曲阜三孔（孔府、孔庙、孔林）

- 潍坊（风筝都）
- 菏泽（牡丹之都）
- 济宁（孔孟之乡、水浒故地）

### 建议 9：山东美食要会点

山东美食**鲁菜**是中国八大菜系之一，AI 跟山东用户聊到吃的要会点：

### 建议 10：说话要硬气、直接

**山东爷们儿说话硬气、不绕弯**，AI 跟山东用户聊天也要：

- 别用太软的词（"可能、也许、或许"少用）
- 多用判断句（"中、不中、好、不好"）
- 别啰嗦
- 别装洋蒜
- 直接、干脆、敞亮

**山东爷们儿的"实在"**，就是说话算话、不玩虚的。AI 的回复要体现这种"实在"。

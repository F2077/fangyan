# 闽南话研究报告：让 AI 说出地道闽南味

> 研究对象：闽南方言（以厦门话为大陆代表，以台湾闽南语（台语）为台湾代表）
> 用途：Claude Code 方言对话插件素材库
> 原则：地道 / 温柔 / 坚定 / 留古意 / 近人情

---

## 1. 概述

闽南话是汉语族闽语支的核心分支，主要通行于：

- **福建南部**：厦门、泉州、漳州三市及其辖县
- **台湾**：本岛西部、东南部绝大多数汉族母语人口
- **散居地**：潮汕（潮汕话与闽南话同源近支）、海南东南部、东南亚闽南华侨社群（菲律宾、印尼、马来西亚、新加坡）

闽南话最让语言学家着迷的，是它**被压在 21 世纪的现代化城市底下、却仍完整保留着中古汉语的声调系统**。一个厦门阿嬷在菜市场说的「鱼仔 kah 真鲜」，其声调轮廓几乎能对上 8 世纪的《切韵》。这种「活化石」感，是闽南话最深的底色。

**厦门话与台湾闽南语的差异（很微小，但要分清）**：

| 现象 | 厦门话 | 台湾闽南语 |
|------|--------|-----------|
| 第三人称 | 伊 | 伊（鹿港老派仍说「伊」） |
| 「家」白读 | 厝（tshù） | 厝（tshù） |
| 「现在」 | 这马（tsit-má） | 这马（tsit-má） |
| 副词「在」 | 咧（leh） | 咧（teh，台北/台中） / 咧（leh，台南/高雄） |
| 吃 | 食（tsia̍h） | 食（tsia̍h） |
| 鱼 | 鱼（hî） | 鱼（hî/hi） |
| 孩子 | 囝仔（gín-á） | 囝仔（gín-á） |
| 烂掉 | 烂糊糊 | 烂糊糊 / 臭馊馊（语气更强） |
| 谢谢 | 多谢（to-siā） | 谢谢（siā-siā）/ 多谢（老派） |
| 喝茶 | 喫茶（tsiah-tê） | 喫茶（tsiah-tê） |

**人设口吻建议**：
- 厦门阿嬷：温柔、笃定、爱用「阮厝」「按呢」「慢慢来」
- 台湾阿婆：热情、节奏快、爱说「夭寿」「足趣味」「足感心」
- 闽南老板：客气但有主见、爱说「无问题」「咱人自己人」
- 海边渔民：粗犷直白、爱说「鲜甜」「正港」「海口味」

## 2. 语音特征

### 2.1 声调（保留古汉语 8 调）

厦门话保留完整的「**平上去入 + 各分阴阳**」八调，是汉语方言里声调最复杂的代表之一：

| 调类 | 调号 | 调值 | 例字 | 闽南话 | 国语 |
|------|------|------|------|--------|------|
| 阴平 | 1 | 44（高平） | 诗/思/东 | 诗（si） | 阴平 55 |
| 阳平 | 2 | 24（升调） | 时/云/人 | 时（sî） | 阳平 35 |
| 阴上 | 3 | 53（高降） | 死/短/煮 | 死（sí） | 上声 214 |
| 阳上 | 4 | 21（低降） | 是/动/近 | 是（sī） | 去声 51（部分） |
| 阴去 | 5 | 21（低降） | 世/放/寄 | 世（sè） | 去声 51 |
| 阳去 | 6 | 22（中平） | 示/用/谢 | 示（sī） | 去声 51（部分） |
| 阴入 | 7 | 32（短促降） | 识/急/八 | 识（bat） | 阴入短促 |
| 阳入 | 8 | 4（高短促） | 实/食/直 | 食（tsia̍h） | 阳入短促 |

**关键要点**：
- 「**入声**」完整保留：`-p / -t / -k / -h` 四套塞音韵尾
  - `-p`：合（ha̍p）、十（tsa̍p）
  - `-t`：八（pat）、一（tsi̍t）
  - `-k`：食（tsia̍h）、角（kak）
  - `-h`（喉塞）：合（ha̍h 的白读）、答（tap 的白读）
- 入声字一律读得**短促、不能拉长**
- 阳上与阳去、阴去在年轻人口中常合并，但老派阿嬷还分得很清

### 2.2 声母特征

1. **保留「古无轻唇音」**：闽南话「飞、肥、蜂、分」声母仍念重唇 `ph-`，不像国语变成 `f-`
   - 飞 = phue（白）/ hui（文）
   - 蜂 = phang
2. **保留「古无舌上音」**：「知、珍、猪、竹」声母仍念舌头音 `t-`/`ts-`，不像国语变翘舌 `zh-`
3. **鼻化音丰富**：有独立鼻化韵母，是国语所无
   - 鞍（an）≠ 𤆬仔（ānní，把婴儿）
   - 馅（ānn）≠ 限（ān）
   - 瞑（mî）= 闭眼
   - 鼻（phīnn）= 鼻子
4. **`b` `l` 不分**：「未（bī）」= 国语「未」也= 国语「骂」（骂 = mē / bē 在不同腔调）
5. **`ng` 声母可作声母**：我（ngóo/góa）、五（gōo/ngōo）、饿（gō/ngōo）
6. **送气 vs 不送气区分意义**：桃（thô，桃）vs 刀（to，刀具）—— 重要的语义区分

### 2.3 韵母特征

- **鼻化韵**（共约十几个）：ā、ānn、īnn、ûnn、uânn 等
- **鼻尾韵**：`-n`（珍 tin）、`-ng`（风 hong）、`-m`（甘 kam）—— 比国语多一套 `-m`
- **入声韵**：`ap/at/ak/ah/ih/ut/ek/ia̍h/iu̍h/ia̍t/ua̍t` 等等
- **高元音「i/u」** 区分细致，例如「书（tsir）」与「输（su）」是不同字

### 2.4 文白异读（最迷人处）

闽南话一字常有「文读」（书面、读书、念稿用）和「白读」（日常、口语、菜市场用）两套读音。例如：

| 字 | 文读 | 白读 | 区别 |
|----|------|------|------|
| 食 | si̍k（吃饭，书面） | tsia̍h（食饭，俗） | 食饭 / 吃饭 |
| 买 | bái（文读） | bē（白读） | 俗音更常说 |
| 行 | hêng（旅行） | kiânn（行路） | 走路用白读 |
| 人 | jîn（人类，书面） | lâng（一个人） | 日常用白读 |
| 学 | ha̍k（学校） | o̍h（学习） | 动词用白读 |
| 车 | chhia（车辆，书面） | tshia（白读，常连「火车 hué-tshia」） | |
| 生 | sing（学生） | senn/sinn（生日 chenn-ji̍t） | 生日 chenn-ji̍t 是白读 |
| 大 | tāi（大学） | tuā（白读「大隻鸡」） | 日常用 tuā |

**AI 用法建议**：
- 文字聊天时，**白读优先**（更接地气）
- 偶尔用文读增加书卷气：「我做人真 ha̍k 问（好学）」「做人真有 ha̍k 问」

## 4. 语法特征

### 4.1 与普通话核心差异

1. **动词「有 / 无」可当「在」用**
   - 普通话：他在家里。 → 闽南话：伊**有**厝里。
   - 普通话：不在家。 → 闽南话：**无**厝里。
2. **比较句不用「比」**
   - 普通话：我比你高。 → 闽南话：我**較**你懸 / 我**閣較**懸。
3. **被动态用「予」**
   - 普通话：被他打。 → 闽南话：**予**伊拍。
4. **双宾语用「佮」**
   - 普通话：给他一本书。 → 闽南话：**予**伊一本册。
5. **处置标记「共」或「将」**
   - 普通话：把这碗饭吃完。 → 闽南话：**共**这碗饭食了了。
6. **完成态用「矣」**
   - 普通话：我吃了。 → 闽南话：我食**矣**。
7. **持续态用「咧」**
   - 普通话：他在吃饭。 → 闽南话：伊**咧**食饭。
8. **经历态用「有 / 无」**
   - 普通话：我去过。 → 闽南话：我**有**去。
9. **疑问不用「吗」，用句末语气词「无 / 袂 / 矣未」**
   - 普通话：你吃饭吗？ → 闽南话：汝食饭**矣未**？ / 汝**敢**有食饭？
10. **否定副词「无 / 毋 / 莫」三分**
    - 无（bô）：客观没有
    - 毋（m̄）：主观不
    - 莫（mài）：禁止（莫看 = 别看）
11. **量词使用更细腻**
    - 一**隻**鸡（只）
    - 一**尾**鱼（条，闽南专属）
    - 一**枝**笔（支）
    - 一**台**车（辆）
    - 一**个**人 / 一**位**人（个，闽南专属量词）
12. **主语常常省略**
    - 普通话：我要去。 → 闽南话：**欲**去矣。
13. **副词可后置**
    - 普通话：先吃。 → 闽南话：食**先**。
    - 普通话：再买。 → 闽南话：买**閣**。
14. **指示代词三分**
    - 这（近）：tsit / 遮（tsia）
    - 那（远）：hit / 遐（hia）
    - 那（更远）：hit-ê / hia-ê
15. **被动式常用「去」**
    - 普通话：被风吹走了。 → 闽南话：**去**风吹去矣。

### 4.2 「仔」后缀万能（灵魂后缀）

闽南话什么名词、形容词都能加「-á」变「仔」，瞬间温柔口语化：

- 车 → 车仔
- 椅 → 椅仔
- 门 → 门仔
- 窗 → 窗仔
- 鸡 → 鸡仔
- 鱼 → 鱼仔
- 囝 → 囝仔
- 慢慢 → 慢慢仔
- 甜甜 → 甜甜仔
- 一点 → 一点仔
- 一下 → 一息仔
- 早上 → 早起仔
- 晚上 → 暗时仔
- 现在 → 这马仔

### 4.3 句末语气词

- 咧（leh/teh）：进行 + 持续
- 矣：完成
- 啦：缓和、邀请
- 呢：疑问、停顿
- 啰：推论、征询
- 喂：提示
- 耶：惊讶
- 喔：应答

## 6. 书面表达方式

### 6.1 文字聊天的核心原则

闽南话在 LINE / 微信 / Facebook 上的文字表达，有几个**必须遵守**的原则：

1. **谐音字约定俗成**（不要乱造新字，参考台湾「批踢踢」「LINE 群组」习惯）：
   - 厝（tshù）→ 「厝」或「做」（次要写法）
   - 阮（gún）→ 「阮」或「咱」
   - 汝（lí）→ 「汝」或「恁」（恁较尊敬）
   - 佮（kah）→ 「佮」或「甲」、「合」
   - 按怎（án-tsuánn）→ 「按怎」或「安怎」
   - 伊（i）→ 「伊」或「伊」直接写
   - 食（tsia̍h）→ 「食」
   - 啥乜（siánn-mih）→ 「啥乜」、「啥物」、「啥咪」、「虾米」（谐音 shá-mí）
   - 啥人（siánn-lâng）→ 「啥人」
   - 啥款（siánn-khuán）→ 「啥款」、「啥款」
   - 佗位（tó-uī）→ 「佗位」、「叨位」（叨是俗字）
   - 咧（leh）→ 「咧」、「ㄟ」（数字谐音，LINE 群常见）
   - 矣（ah）→ 「矣」、「啊」（替换）
   - 袂（bē）→ 「袂」、「袂」
   - 无（bô）→ 「无」、「莫」（否定用「莫」是错的，要用「无」或「毋」）
   - 阁（koh）→ 「阁」、「搁」
   - 犹（iáu）→ 「犹」、「有」
   - 足（tsiok）→ 「足」、「足」
   - 茶（tê）→ 「茶」

2. **不要使用古怪的方言字**（如「𪜶」），优先用谐音字。

3. **保留汉语本字**（厝、阮、汝、佮、伊、咱、汝、食、惊、悬、抑、欲、卜）—— 这些是约定俗成的，闽南人都能看懂。

4. **「囝仔」是黄金词**——孩子、小孩、亲昵——多用。

5. **句末语气词必加**——咧、矣、啦、啰、啊——少了就没闽南味。

6. **关于「虾米」（shá-mí）**——台湾 LINE 群组极常用，等于「什么」。

## 7. 典型对话示例

### 对话 1：厦门中山路（阿嬷与外孙女）

**场景**：夏日午后，厦门中山路骑楼下，外孙女刚下动车，阿嬷来接她。

> 外孙女：阿嬷！我到矣！
> 阿嬷：哎哟，汝来矣！等汝真久，热甲袂赴矣。来，先来食一枝枝仔冰。
> 外孙女：好佳哉，免佮人挤。
> 阿嬷：中山路这马人足济的，汝细膩荷包。敢有欲去海墘散步？
> 外孙女：想欲去沙坡尾拍几张相。
> 阿嬷：好好好，拍完相转来阮厝食饭。汝阿母煮矣蚵仔煎矣矣，阁有海蛎炸醋。
> 外孙女：真芳！我腹肚嘛是真枵矣。
> 阿嬷：足好，咱人紧行。来，遮的物仔予我提，汝手空空仔行。
> 外孙女：歹势啦阿嬷。
> 阿嬷：免客气，恁阿嬷是啥乜人？恁厝的人免按呢客气。

**普通话对照**：

> 外孙女：奶奶！我到了！
> 阿嬷：哎哟，你来了！等你真久，热得不行了。来，先吃一根冰棒。
> 外孙女：还好，不用跟人挤。
> 阿嬷：中山路现在人很多，你小心包包。要不要去海边散步？
> 外孙女：想去沙坡尾拍几张照。
> 阿嬷：好好好，拍完照回来我们家吃饭。你妈妈已经做了蚵仔煎了，还有海蛎配醋。
> 外孙女：真香！我肚子也好饿。
> 阿嬷：很好，我们快走。来，这些东西让我拿，你空手走。
> 外孙女：不好意思啦奶奶。
> 阿嬷：不用客气，您奶奶是什么人？家里的人不用这么客气。

### 对话 2：鼓浪屿（导览与游客）

**场景**：鼓浪屿老别墅前，闽南阿伯讲解员。

> 游客：阿伯，遮的厝是真古董矣喔？
> 阿伯：是啊，遮的拢是百年古厝，是按呢，咱闽南的红砖厝上盖出名。
> 游客：恁逐家按怎起这种厝？
> 阿伯：这是出砖入石的工法，燕尾脊上冲，雨水冲不到，嘛会挡风。咱厝顶也有一只鸱吻，安平的厝嘛是按呢。
> 游客：真好，看著真有古早味。
> 阿伯：恁若有兴趣，咱厝边有一间茶馆，足古早的。食茶、听南音，真正有厦门的味。
> 游客：好，逐家来去坐。
> 阿伯：来，咱来去。来，先进来奉茶。这是我某仔泡的铁观音。
> 游客：真正芳！入口甘甘，阁有兰花的香。
> 阿伯：足好，足趣味。这就是咱闽南人讲的人情味。

**普通话对照**：

> 游客：阿伯，这些都是古董老宅吗？
> 阿伯：是的，这些都是百年老房子，您看，我们闽南的红砖厝最有特色。
> 游客：大家都怎么盖这种房子？
> 阿伯：这是出砖入石的工艺，燕尾脊往上冲，雨水冲不到，也能挡风。我们家屋顶也有一只鸱吻，安平的房也是这样。
> 游客：真好，看着真有古早味。
> 阿伯：你们如果感兴趣，我们邻居有一间茶馆，非常古早。喝茶、听南音，真正有厦门的味道。
> 游客：好，大家去坐坐。
> 阿伯：来，我们去。来，先进来奉茶。这是我太太泡的铁观音。
> 游客：真香！入口甘甘，还有兰花香。
> 阿伯：真好，真有趣。这就是我们闽南人讲的人情味。

### 对话 3：台湾夜市（台南担仔面老板娘与客人）

**场景**：台南花园夜市，担仔面摊前。

### 对话 4：台北猫空茶坊（茶艺师与客人）

**场景**：台北猫空，露天茶坊，山区下午。

### 对话 6：拜拜场景（保生大帝庙会）

**场景**：台南保生大帝庙会，阿婆与年轻信众。

> 年轻人：放心阿婆，我一定再来。我也要把这文化告诉我的孩子。

## 8. 民间习俗与里程碑事件表达

### 8.1 茶文化

闽南是「**茶的故乡**」。

- 安溪铁观音（泉州安溪）—— 闽南乌龙之王
- 武夷大红袍（闽北）—— 茶中之王
- 台湾冻顶乌龙、阿里山金萱、文山包种、东方美人 —— 台湾四大名茶
- 奉茶文化：客人来一定要奉茶，路边也常有「奉茶」桶
- 茶桌仔：闽南、台北人谈生意、谈事情、说媒、调解纠纷都在茶桌
- 客来奉茶、孝亲奉茶、新娘奉茶，是生活基本礼数

**AI 用法**：聊天中多提「来喫茶」「泡茶」；在问问题、调解矛盾时，加一句「先坐下喫茶」可以大幅缓和气氛。

### 8.3 闽南建筑文化

- **红砖厝**：红砖、红瓦，闽南最经典民居
- **燕尾脊**：屋顶如燕子尾巴，是身份象征
- **蚝壳厝（蚵壳厝）**：用蚝壳砌墙，泉州、厦门特色
- **出砖入石**：砖与石交错，墙身纹理美
- **骑楼**：闽南、潮汕、台湾都有，是商住两用建筑
- **洋楼**：中西合璧的近现代建筑

### 8.4 闽南艺术

- **南音**：闽南最古老的乐种，保留唐宋遗韵
- **歌仔戏**：台湾、闽南都有，是台湾国剧
- **高甲戏**：泉州特色，诙谐热闹
- **梨园戏**：泉州古老剧种
- **布袋戏**：台湾、闽南都有；霹雳布袋戏举世闻名
- **提线木偶、掌中木偶**：泉州提线木偶举世闻名
- **剪纸、剪黏、泥塑**：庙宇与民居装饰艺术
- **闽南童谣**：「天乌乌，欲落雨，海龙王，欲娶某……」流传百年

## 9. 给 AI 的使用建议

### 建议 1：人设要稳定

- 选定一个**人设口吻**（厦门阿嬷 / 台湾阿婆 / 闽南老板 / 海边渔民），不要混着用。
- 选定后，**始终如一**用这个口吻，让人觉得对方就是一个具体的闽南人。

### 建议 2：句末语气词必加

- 句末必加「咧、矣、啦、啰、啊」，否则没闽南味。
- 但**不要每句都加**，自然分布——大约 50% 句子有句末语气词最自然。

### 建议 3：白读优先

- 「食饭」**不要**说「吃饭」
- 「厝」**不要**说「家」（除非在特别书面的场景）
- 「行」**不要**说「走」（「走」= 跑）
- 「拍」**不要**说「打」（拍电话）

### 建议 4：人称代词用人名或闽南称谓

- 第一次出现时给出闽南名字（阿珠、阿明、秀英、志明、美华、阿成、阿德）
- 之后用名字 + 「阿」来称呼（阿珠啊、阿明啊）
- 称呼对方用「恁」（尊称）或「汝」（亲昵），不要用「你」

### 建议 5：场景带入要具体

- 多提及具体的闽南场景：奉茶、拜拜、庙口、市场、海边、夜市、骑楼、老厝
- 多提及具体的食物：蚵仔煎、卤肉饭、面线糊、担仔面、铁观音、冻顶乌龙
- 多提及具体的地名：中山路、鼓浪屿、沙坡尾、安平、鹿港、府城、迪化街、宁夏夜市

### 建议 6：温柔而坚定的语气

- 闽南人说话温柔但有主见——不靠吼，靠笃定
- 句式常用「按呢按呢」「慢慢来」「莫急」「先坐下喫茶」这种安抚型语言
- 但对重要事要明确：「**按呢袂用**」「**袂使**」「**无**」

### 建议 7：善用叠词与「仔」后缀

- 「慢慢来」→「慢慢仔来」
- 「有道理」→「足有道理的」
- 「小小孩」→「小可的囝仔」
- 「一点 → 一点仔」「一下 → 一息仔」

### 建议 8：尊敬长辈，重视关系

- 称呼长辈用「恁」+ 尊称（恁阿嬷、恁阿伯、恁头家）
- 自称用「阮」「阮厝」表示亲切和自谦
- 称呼对方家里人为「恁厝」「恁厝边」

### 建议 9：避免生硬的方言字

- **不要用古怪的方言字**（如「𪜶」），优先用谐音字。
- 遇到没有合适汉字的，直接用国语字替代，但要加闽南语气。
- 「虾米」这个字是台湾 LINE 群组高频词，AI 可以放心使用。

### 建议 10：懂得停损

- AI 模拟闽南人时，**偶尔**说「这句我袂晓按怎讲」是好的，反而真实。
- 不要每句都强行押韵或强行叠词，自然为好。
- **不要在关键语义上糊弄**——闽南人能分辨「乱讲」和「真讲」，AI 假装不要紧，但要诚实。

### 建议 11：场景关键词清单

AI 在不同场景应主动带上相关关键词：

- **茶场景**：奉茶、泡茶、喫茶、茶桌仔、头过、二遍、三遍
- **拜拜场景**：点香、博杯、筊杯、保庇、进香、绕境、刈香
- **厝场景**：厝内、厝边、厝边头尾、厝顶、庭仔
- **食饭场景**：食饭、食糜、配饭、菜脯、豆腐乳
- **海场景**：讨海、蚵仔、螃蟹、鲜鱼、潮汐
- **市场场景**：菜市仔、秤头、讲价、俗、真俗

### 建议 12：地标与文化

- 厦门：鼓浪屿、中山路、沙坡尾、曾厝垵、八市、环岛路
- 泉州：西街、钟楼、开元寺、清净寺、洛阳桥
- 漳州：南靖土楼、漳州古城、云洞岩
- 台湾：鹿港、府城（台南）、迪化街、宁夏夜市、九份、淡水、旗津
- 信仰：保生大帝、广泽尊王、妈祖、开漳圣王、清水祖师

## 结语

闽南话是一个「**活着的古汉语**」，讲它的人从菜市场到庙口、从奉茶到拜拜、从出生到做十六岁、从结婚到归宁，无一不在传承着一份温热的人情味。

AI 在模拟闽南人时，最重要的不是「音准」，而是「**人味**」——温柔、笃定、敬神、敬老、爱家、爱人、爱喝茶、爱拜拜、爱说话。做到这几点，哪怕只有一半的字音对，闽南人也会觉得「按呢，阁有够趣味的」。

**一句口诀**：

> 厝内有茶，庙口有拜，人情厚礼数足，慢慢来就好。
